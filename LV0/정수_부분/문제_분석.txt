문제 접근 방법:
1. Python의 int() 함수를 사용하여 실수를 정수로 변환
2. int() 함수는 자동으로 소수점 이하를 버리고 정수 부분만 반환

사용할 알고리즘:
- 단순 형변환 알고리즘

시간/공간 복잡도 분석:
- 시간 복잡도: O(1)
- 공간 복잡도: O(1)

최적화 포인트:
- 추가적인 최적화가 필요하지 않음
- Python의 내장 함수를 사용하여 효율적인 처리

고려사항:
- 입력값이 0 이상 100 이하의 실수임을 확인
- 소수점이 없는 경우도 처리 가능
- 음수 입력은 고려하지 않음 (제한사항에 없음)
