알고리즘 구현 단계:
1. solution 함수 정의
   - 매개변수: s (str)
   - 반환 타입: str
   - Type hints와 docstring 추가

2. 핵심 로직
   - swapcase() 메서드를 사용하여 대소문자 변환
   - 변환된 문자열 반환

3. 테스트 케이스 구현
   - 정상 케이스: 'aBcDeFg'
   - 경계값: 'a', 'Z'
   - 추가 테스트: 'HelloWorld'

예외 처리:
- 입력값 범위 검증 (1 ≤ len(s) ≤ 20)
- 알파벳 문자열 검증
